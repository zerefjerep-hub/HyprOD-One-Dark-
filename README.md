# HyprOD OneDark (Masih dalam pengembangan)
Harusnya tinggal launch ./install.sh kelar.

untuk wallpaper bisa clone git repo ini :<br>
<pre>
git clone https://github.com/AbdelrhmanNile/onedark-wallpapers --depth=1</pre><br><br>
Terus ketik ini di terminal:<br>
<pre>
awww img ./onedark-wallpapers/pilihFileGambar.formatnya --transition-type center</pre><br>

Rekomendasi Wallpaper :<br><br>
<pre>
awww img ./onedark-wallpapers/misc/od_clouds.png --transition-type center<br>
awww img ./onedark-wallpapers/misc/od_plane.jpg --transitio-type center
</pre>

Kalo tulisan di terminal kecil lakuin ini :<br>
nvim ~/.config/kitty/kitty.conf<br>
ubah angka font_size<br>

# MAIN SHORTCUT
<pre>
Terminal (kitty)       = Super + Enter
File Explorer (Thunar) = Super + E
Quit App               = Super + Q
App Launcher           = Super + D
Screenshot             = PrtScr
Reset Waybar           = Super + R
Floating Window        = Super + V
Move Window            = Super + LMB
Resize Window          = Super + X
Workspace              = Super + 1, 2, 3, ...
Next Workspace         = Super + Right Arrow
Previouse Workspace    = Super + Left Arrow
Shutdown               = Super + D -> cari Shutdown
Reboot                 = manual pake terminal (sudo reboot now)
</pre>
